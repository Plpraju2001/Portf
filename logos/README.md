# Organization Logos

Place your organization logo files in this directory with the following names:

## Education Logos
- `clark-university-logo.png` - Clark University logo
- `jntuk-logo.png` - JNTUK logo

## Experience Logos
- `unitedhealth-group-logo.png` - UnitedHealth Group logo
- `brane-enterprises-logo.png` - Brane Enterprises logo

## Notes
- Recommended size: 200x200px to 400x400px
- Supported formats: PNG, JPG, SVG
- The code will automatically fallback to external URLs if local images are not found
- Ensure logos maintain aspect ratio and are clear/readable
