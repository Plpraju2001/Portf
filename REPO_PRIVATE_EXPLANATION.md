# What Those Effects Mean

## Reading the GitHub Modal

When you click "Make private", GitHub shows some effects. Here's what they mean:

### 1. "Custom Dependabot alert rules will be disabled"
**What it means:** If you had special security alerts set up, those will stop
**Should you care?** No, for a portfolio site this doesn't matter
**Your action:** You can ignore this

### 2. "Current forks will remain public and will be detached"
**What it means:** If anyone forked your repo, their forks stay public but won't get updates from you
**Should you care?** Only if someone forked it (unlikely for a portfolio)
**Your action:** You can ignore this

### 3. "Advanced Security will be disabled"
**What it means:** GitHub's premium security features turn off
**Should you care?** No, unless you were paying for advanced security
**Your action:** You can ignore this
