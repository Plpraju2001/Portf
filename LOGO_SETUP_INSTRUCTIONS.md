# Logo Setup Instructions

To add genuine company logos to your portfolio:

## Step 1: Download Logo Files

1. **Scale AI Logo:**
   - Visit: https://scale.com (check their press/media kit)
   - Or download from: https://logo.clearbit.com/scale.com
   - Save as: `scale-ai-logo.png`

2. **American Express Logo:**
   - Visit: https://www.americanexpress.com (check their brand guidelines)
   - Or download from: https://logo.clearbit.com/americanexpress.com
   - Save as: `american-express-logo.png`

## Step 2: Add Logo Files

1. Place the downloaded logo files in: `public/logos/`
2. Make sure the files are named exactly:
   - `scale-ai-logo.png`
   - `american-express-logo.png`

## Step 3: Verify

After adding the files, the portfolio will automatically use them instead of external URLs.

## Alternative: Use Clearbit Logo API

The current setup uses Clearbit Logo API which provides genuine company logos:
- Scale AI: https://logo.clearbit.com/scale.com
- American Express: https://logo.clearbit.com/americanexpress.com

These should display automatically once deployed.
